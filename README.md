# SkillSync AI 🚀
### Smart Resume Analyzer & Job Matcher

## Live Demo
👉 https://skillsync-ai-two.vercel.app

## About
SkillSync AI is a full-stack web application that analyzes your resume and matches it with your desired job role using AI.

## Features
- 📄 Upload Resume (PDF)
- 🎯 Match Score (%)
- ✅ Skills Found
- ❌ Missing Skills
- 💡 AI Suggestions to improve

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js + Express.js
- **Database:** MongoDB Atlas
- **AI:** Groq (Llama 3.3)
- **Deployment:** Vercel

## How to Run Locally

### Backend
cd backend
npm install
npm run dev


### Frontend
cd frontend
npm install
npm start


## Environment Variables
Create a `.env` file in backend folder:
PORT=5000
MONGO_URI=your_mongodb_uri
GROQ_API_KEY=your_groq_api_key
